# LeetCode2Github

LeetCode to Github

Automatically grab the LeetCode solutions uploaded to Github and generate `README.md` file.

`run.py`

`leetcode.py`

`question.py`

`utils.py`

`generate.py`



配置信息

- `HOME`   
- `LOCAL_PATH`
- `REPO`  GitHub repo地址
- `USER`   用户 leetcode 登陆账号
- `PASSWD`  用户 leetcode 登陆密码
